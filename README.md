# 键鼠助手 v2.1

这是一个简单的 Windows 自动化小工具，主要帮我处理一些重复的点击和按键任务。

### 核心功能
* **进程绑定**：可以指定在某个程序里运行，也可以留空全局使用。
* **按键映射**：自己设定触发键，支持加个延迟（比如 100ms）再触发下一个键。
* **鼠标控制**：设定好坐标就能自动点，还能用“捕获”功能直接抓取当前的鼠标位置。
* **自然模拟**：
    * **时间随机**：点按间隔可以加个随机范围，避免死板的固定频率。
    * **坐标抖动**：在选定的像素范围内随机晃动着点，更像真人在操作。
* **总是置顶**：勾选后窗口不会被挡住，方便随时改参数。
* 请以管理员模式启动

KeyMouse Assistant v2.1 - Simple Automation Tool

A lightweight tool I built to automate repetitive clicks and keystrokes.

Process Binding: Target a specific app or leave it blank for global use.

Key Mapping: Custom triggers with adjustable delays between actions.

Mouse Control: Set coordinates manually or use the "Capture" button to grab your cursor position.

Natural Simulation:

Random Interval: Adds a random time buffer to avoid robotic, fixed frequencies.

Spatial Jitter: Shifts the click slightly within a pixel range to mimic human movement.

Always on Top: Keeps the tool visible for quick adjustments.

This tool requires administrator privileges to function correctly.

Note: Keep the EXE and INI files in the same folder for the configuration to work.
